# SCT_SD_4
E-commerce Product Scraper (Python + Tkinter GUI) A desktop application that extracts product information such as name, price, and ratings from e-commerce webpages using web scraping (BeautifulSoup + Requests). The application features a clean Tkinter GUI, displays extracted data in a table, and allows exporting results to a CSV file.
